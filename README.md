# D3 Intro Bar Chart

A sample using D3js v3 library to render a bar chart using DIV elements.

## Installation
*	Open Terminal
* cd to 'd3-intro' folder
* Type `npm install'

## Running locally
* In terminal or command prompt; type:
* `gulp`
* hit enter


### Note:
If you get errors on libsass, then run
`npm rebuild node-sass`
